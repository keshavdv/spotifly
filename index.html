Spotify Connect with Node.js

Download .zip Download .tar.gz View on GitHub

Where's the Magic @?

Spotifly allows you to turn any device into a Spotify Connect enabled endpoint. Simply start the spotifly server and the device will show up on your phone as a Connect device

$ npm install -g spotifly 
$ spotifly -u <username> -p <password> -n <device_name>
